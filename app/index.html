Using Yeoman

A better way to build a website

Today we will learn how to integrate Yeoman into your workflow.


First, what is Yeoman? Yeoman, in a nutshell, is three different tools: yo, bower, and grunt. Yo is a frameworks tool that when launched, builds the skeleton of your future website. However, Yo alone just produces a nice-looking and ready-made website.


Bower then combs github and other repositories for the necessary libraries, and integrates them into your website.


Finally, grunt is used to support this whole process. Grunt gives you immediate access to a virtual server for live output, testing options through Karma, and added functionality to condense your code and files down for faster read/write.
